Dataset(s)

Ambiguously Plural

I plan to use the UNESCO Atlas of the World's Languages in Danger dataset (last updated 2015, accessed April 2020).

The Terms of Service page linked directly on the website is no longer accessible, but as this dataset was created by a UN organization, it is free for personal and non-commercial use, as the Internet Archive confirms.

I have filled out the form to request access to the full version of the dataset (which includes location data); in the meantime, I am using a 2011 version published by the Guardian.

This dataset contains 2723 rows (languages) and 15 columns (though not all cells have values, and some languages are missing speaker population data) Relevant columns include:

  • ISO Language Codes: A standardized reference for other sources
  • Language Names: In English, French, Spanish, etc
  • Countr(ies) Spoken: A list.
  • Location Description: More information about where in those countries the language is spoken
  • Degree of Vitality: Vulnerable, Definitely endangered, Severely Endangered, Critically Endangered, Extinct (I would like further information about the distinctions between these categories)
  • Number of Speakers: from 0 to 7500000 (South Italian, which I personally would not have included).
  • Sources: Where the information was comipled from
  • Latitude & Longitude: Unfortunately a point, not a region, which woul dbe more accurate

If necessary, I may also supplement this dataset with information from Glottolog, which includes information about the world's language, language family/child dialects in hierarchical structure, and macro-area (available under Creative Commons License) and The World Atlas of Language Structures Online, which provides information about internal language features (also availble under Creative Commons License). Unlike the UNESCO dataset, neither of these are limited to endangered languages, so I didn't want to use them as my main dataset since there's just too much information (Glottolog has 8,500 entries!), but they may be useful for cross-referencing or adding hierarchical features.
